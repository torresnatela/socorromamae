# Socorro Mamãe — ByeBye Fralda

Protótipo responsivo criado em Next.js para o fluxo "Socorro Mamãe — ByeBye Fralda", com telas de cadastro, chat com IA, painel de progresso e diretrizes conversacionais.

## Como executar

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
3. Abra [http://localhost:3000](http://localhost:3000) no navegador para visualizar o layout.

## Scripts disponíveis

- `npm run dev` — executa o modo de desenvolvimento com recarregamento automático.
- `npm run build` — gera a versão de produção otimizada.
- `npm run start` — inicia o servidor em modo de produção.
- `npm run lint` — roda as verificações de lint padrão do Next.js.
